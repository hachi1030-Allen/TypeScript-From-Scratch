# Core Syntax and Features

## Core Types

- number
  - TS only has one type for number, including all intergers, floats, etc.
  - Example: 1, 5.3, -10
- string
  - All text value
  - Example: 'Hi', "Hi", \`Hi\`
- boolean
  - Only has `true` and `false`, no "truthy" or "falthy" values
  - Example: true, false
